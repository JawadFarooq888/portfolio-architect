# Modern Developer Portfolio

This is a high-end, data-driven portfolio designed for a multi-stack developer specializing in:
- Android (Kotlin)
- Flutter
- Data Engineering & ETL
- Database Architecture

## Features
- **Modern UI:** Built with Tailwind CSS and Lucide Icons.
- **Dark Theme:** Professional slate-based color scheme with cyan/emerald accents.
- **Data-Driven:** Projects and skills are managed via `data.js`, making it easy to update without touching HTML.
- **DB Simulation:** The contact form saves data to `localStorage` to demonstrate database interaction logic.

## How to Run
1. Navigate to the `portfolio_web` folder.
2. Open `index.html` in any web browser.

## Customization
- **Edit Projects:** Open `data.js` and modify the `portfolioData.projects` array.
- **Change Colors:** Update the Tailwind classes in `index.html`.
- **Real Database:** To connect a real database, I recommend integrating **Supabase** (PostgreSQL) which aligns perfectly with your Database Developer background.
